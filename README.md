# CS3219 Project (PeerPrep) AY2526S2 - Group 09

PeerPrep is a technical interview preparation platform where users can match with peers based on topic and difficulty to solve coding problems in real-time.

## Quick Start

### Prerequisites

- [Docker Desktop](https://www.docker.com/products/docker-desktop/) installed and running.
- [Node.js v20+](https://nodejs.org/) (for local linting/testing).

1. Clone the repository and navigate to the root directory of the project.
2. Run the following command to build and start the services:

```bash
   docker-compose up --build
```

3. The frontend will be available at `http://localhost:3000`.
